Welcome to wp-api-angularjs.

WordPress WP-API(v2) services for AngularJs

So far the lib covers 100% of GET methods for all entities.

This library only supports basic auth. OAuth1 not being suitable for JS clients (it would mean exposing key and password out of the open)

This project is written in ES6

Wp-api-angularjs is maintained by shprink
